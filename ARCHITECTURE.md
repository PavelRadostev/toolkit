Toolkit — набор общих библиотек для микросервисов (МС). Основные цели:
- унификация взаимодействия через Bus
- единый подход к конфигурации
- единые интерфейсы и утилиты

## Структура проекта
- `/cmd` — примеры использования toolkit
- `/pkg` — публичные пакеты, доступные другим микросервисам
- `/internal` — реализация интерфейсов, скрытая от внешнего кода

## Bus
- `/pkg/bus`
- `/internal/bus/redis_bus` — реализация через Redis

## Config
- `/pkg/config` — метод получения структуры с настройками для МС
- `/internal/config` — загрузка и маппинг из YAML/ENV

## Принципы
- Публичный API — через `/pkg`
- Логика по работе с Redis — в `/internal`
- События должны быть плоскими DTO-структурами
